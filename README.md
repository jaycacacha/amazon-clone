# Amazon Clone Project

* Demo (https://clone-f6d1b.web.app/)

This project was made with

* React Hooks
* React Router DOM
* React Stripe to collect online dummy payments.
* Firebase Authentication for login and account creation
* Firestore as database
* Flexbox
* Material-ui
* Axios
